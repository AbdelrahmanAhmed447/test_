# HI Git tutorial
# First git init for initializing the git main directory 
# second add files must be in the git main directory in order to be recognized 
# third git add file name to be ready to be committed
# Fourth git commit -m "added file name" to be committed
# Fifth repeat the git add again after modifying any file then the aforementioned git commit -m file name
# sixth  git checkout -b new to branch off from the git master whiling keeping all original content
# Seventh rm -f .git/index.lock to restart after crash
# Eighth git merge master to merge branch with master git although only the branch will have both data while master will be unchanged
# Ninth git remote add origin https://github.com/AbdelrahmanAhmed447/test_.git to link local git to global github
# Tenth git push -u origin main to finalize
